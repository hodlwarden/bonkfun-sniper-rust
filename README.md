## Installation

1. Clone the repository:

```bash
git clone https://github.com/hodlwarden/bonkfun-sniper-rust.git
cd bonkfun-sniper-rust
```
2. Install dependencies (ensure you have Rust installed):

```bash
cargo build --release
```
3. Usage

```bash
cargo run
```
